# Jimmy's Ranch Dog Grooming Salon

A dog grooming salon landing page built on top of the original on-scroll shape morph animations and interactive tilt effect.

This project keeps the animated scrolling experience while reframing the content as a playful pet spa website for Jimmy's Ranch Dog Grooming Salon.

## Installation

Run this demo on a [local server](https://developer.mozilla.org/en-US/docs/Learn/Common_questions/Tools_and_setup/set_up_a_local_testing_server).
